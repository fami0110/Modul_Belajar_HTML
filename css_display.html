1. Apa yang dimaksud dengan property display pada CSS?

  1. Adalah property untuk mengatur tampilan dari element.
  2. Merupakan property untuk menyembunyikan suatu element.
  3. Ialah property untuk mengatur perilaku suatu element saat element ditampilkan.
  4. Semua jawaban benar.

2. Berikut merupakan daftar tag html yang secara default memiliki display: block;, kecuali ...

  1. <p>
  2. <div>
  3. <img>
  4. <h1>

3. Untuk menyembunyikan suatu element, maka syntax css yang perlu kita masukkan adalah ...

  1. display: hidden;
  2. display: disabled;
  3. display: unset;
  4. display: none;

4. Pernyataan yang benar dari display: block; dibawah ini yaitu ...

  1. Element akan berbentuk kotak.
  2. Element akan selalu membentuk baris baru dibawahnya
  3. Width dan height element tidak dapat diatur.
  4. Element akan menutupi element lainnya.

5. Salah satu perilaku dari display: inline; adalah ...

  1. Setiap kali ditambahkan, element tidak akan membentuk baris baru.
  2. Margin dan padding element tidak dapat diatur.
  3. Element tersebut akan sejajar dengan element lainnya.
  4. Susunan element akan seperti garis

6. Dari seluruh element inline, ada salah satu element yang width dan height-nya dapat kita atur, yaitu ...

  1. <span>
  2. <a>
  3. <i>
  4. <img>

7. Ada 2 cara untuk menyembunyikan suatu element, yaitu menggunakan display: none; dan visibility: hidden;. Apa perbedaan dari kedua property tersebut?

  1. display: block; akan menghilangkan element dari tampilan web. Sementara visibility: hidden; akan menyembunyikan element.
  2. display: block; akan menyembunyikan element dari tampilan web. Sementara visibility: hidden; akan menghilangkan element dari susunan html.
  3. display: block; akan menghilangkan element dari tampilan web. Sementara visibility: hidden; juga sama, namun masih menyisakan space yang dihasilkan dari element tersebut.
  4. display: block; akan membuat element tidak terlihat pada tampilan web. Sementara visibility: hidden; membuat element menjadi transparan.

8. Bagaimana caranya agar kita dapat mengatur width dan height dari suatu element, namun perilakunya tetap seperti display: inline;?

  1. Menambahkan display: inline; dan display: block; secara bersamaan.
  2. Menggunakan display: block;.
  3. Menambahkan property width dan height.
  4. Menggunakan display: inline-block;
